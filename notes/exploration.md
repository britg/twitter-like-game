### Exploration

Default Actions:
- Explore
- Observe
- Find
  - list of findable objects based on skills/crafting, e.g. "Lumber, ore"
    - find_survival : adventuring
    - find_herbs : herbalism
    - find_lumber : woodworking
    - find_ore : mining
    - find existing nodes you've discovered in this location
- Craft - See [[Crafting]]
